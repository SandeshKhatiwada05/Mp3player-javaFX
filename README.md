# Sandesh's MP3 Player 🎵
A simple MP3 player built using JavaFX 22. Play, pause, and browse your favorite songs with a modern UI.

🛠 Tech Used: JavaFX, Maven, FXML

📦 Run:
mvn javafx:run

🎧 Features:
- Play/Pause/Stop
- File Chooser to select MP3
- Stylish UI with custom CSS

📁 Requirements:
- Java 22+
- Maven installed

Enjoy the music! 🎶
